Работа с ветками
git branch git clone адрес репозитория - клонировать репозиторий на локальный компютер git pull - скачивается актуальная версия удаленного репозитория и все изменения применяются к локальному репозиторию git push - все последующие разы отправка изменений в удаленный репозиторий
